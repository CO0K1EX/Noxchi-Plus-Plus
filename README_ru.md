# Nohchi++ (Нохчи Плюс Плюс)

> Экспериментальный слой над C++, позволяющий писать код с использованием чеченских ключевых слов.

---

## 📌 О проекте

**Nohchi++** — это экспериментальный проект, который переосмысляет синтаксис C++, делая его ближе к естественному языку (чеченскому).

Проект реализован с помощью макросов и алиасов, которые заменяют стандартные конструкции языка.

---

## 💡 Идея

Цель проекта — исследовать:

* можно ли адаптировать язык программирования под естественный язык
* как меняется читаемость кода
* насколько гибок C++ как основа для DSL

---

## 🔧 Пример кода

```cpp id="example001"
#include "noxchi_plus_plus.h"

БОЛХ_БОЛО

    доллутерахь x = 5;

    нагахьсанна (x > 0)
    {
        хаам(L"Положительное число");
    }
    кхечутайпана
    {
        хаам(L"Отрицательное число");
    }

БОЛХ_ЧЕКХБЕЛИ
```

---

## ⚙️ Как это работает

Проект использует:

* `#define` — для замены ключевых слов (`if`, `while`, `return`)
* `using` — для создания алиасов типов
* стандартную библиотеку C++

---

## 🧠 Особенности

* собственные названия типов (`доллутерахь`, `сиз`)
* локализованные конструкции (`нагахьсанна`, `цкъачунна`)
* упрощённый ввод/вывод (`хаам`, `лаха`)
* автоматическая настройка консоли под Unicode (Windows)

---

## ⚠️ Ограничения

* макросы усложняют отладку
* не предназначено для production
* ограниченная поддержка IDE

---

## 🚀 Использование

1. Подключить заголовочный файл:

```cpp
#include "noxchi_plus_plus.h"
```

2. Использовать макросы вместо стандартного синтаксиса
